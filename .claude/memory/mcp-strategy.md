# MCP Strategy

## Hybrid Approach
1. **Platform-provided MCP tools** - Built and maintained by us
2. **Business-provided MCP tools** - Companies bring their own MCP servers

## MVP - Platform MCP Tools

| Tool | Purpose |
|------|---------|
| RAG Retrieval | Query org's knowledge base |
| Send Notification | SMS/WhatsApp for escalation |
| Create Ticket | Log unresolved issues for async support |
| Knowledge Gap Flag | Notify business about content gaps |

## Future - Pre-built Connectors
- Notion sync
- Google Calendar (appointment booking)
- Email notifications
- Generic API connector

## Future - Business-Provided MCP
- Companies can onboard their own MCP servers
- Validation and sandboxing required
- SSO/auth integration for security
