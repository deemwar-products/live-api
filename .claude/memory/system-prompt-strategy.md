# System Prompt Strategy

## Three Modes

| Mode | How it works | Best for |
|------|--------------|----------|
| **Auto-generated** | AI analyzes uploaded docs → generates system prompt | Quick start, no effort |
| **Auto + Edit** | Auto-generated prompt, user can customize/edit | Balance control + convenience |
| **Custom** | User writes system prompt from scratch | Full control, most token-efficient |

## Benefits
- Small businesses: Just upload docs, AI figures it out
- Cost-conscious businesses: Can edit auto-generated prompt to remove unnecessary parts → save tokens
- Custom businesses: Full control over AI behavior

## Behavior Settings (businesses can also configure)
- Greeting message
- Escalation triggers
- Tone/style preferences
- Behavior rules

## Note
Token/cost optimization - users can trim auto-generated prompts to reduce unnecessary content.
