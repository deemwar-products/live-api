# MVP Success Metrics

## Key Metrics & Targets

| Metric | Target | Notes |
|--------|--------|-------|
| **Automation rate** | 80-90% | Queries resolved by AI without human help |
| **Escalation rate** | <20% | Queries needing human intervention |
| **AI accuracy** | >80% | Judge LLM scores on answer quality |
| **Response latency** | <2 seconds | AI response time target |
| **Customer satisfaction** | TBD | Based on feedback/ratings |

## Additional Metrics

| Metric | Why it matters |
|--------|---------------|
| Time to onboard | How fast can a business go live |
| Knowledge base coverage | % of FAQ topics covered by docs |
| Escalation feedback loop speed | How fast businesses update KB based on feedback |

## Note
Metrics will be refined after production data.
